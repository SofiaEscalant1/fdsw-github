Acerca de mi

Soy una persona perseverante. Me encanta aprender, desarrollar proyectos, escalar y conectar.

Full Stack Developer

Programador junior con manejo de HTML y CSS; competencias técnicas en lenguajes de programación como Python, R, C++ y conocimientos en sistemas de control de versiones como Git. Participé en hackathones con equipos multidisciplinarios, demostrando trabajo en equipo y comunicación efectiva.

Mis trabajos

Desafío 1: Curriculum Vitae

Se creó un currículum vitae en HTML, incluyendo estructura básica, títulos, párrafos, listas, imágenes, enlaces y una tabla de portafolio en una segunda página, siguiendo requerimientos específicos.

Desafío 2: Landing Page

Se construyó un landing page con una sección principal con imagen de fondo, una sección de productos (3 items) y un footer, utilizando CSS externo y estructura HTML válida.

Desafío 3: Landing Page de Iguanas

Se creó una landing page sobre iguanas utilizando CSS Flexbox para el posicionamiento, incluyendo secciones con íconos de Font Awesome, una galería de imágenes y etiquetas semánticas.

Desafío 4: Cuppon

Se desarrolló una página web responsiva para cupones usando Bootstrap, implementando grillas, cards, formularios y clases utilitarias para adaptarse a dispositivos móviles y desktop.
